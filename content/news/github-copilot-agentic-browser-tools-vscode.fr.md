GitHub vient de rendre disponibles en version stable les outils de navigation agentique dans VS Code, permettant à Copilot de parcourir des pages, d'inspecter le contenu, de prendre des captures d'écran et de vérifier que votre appli web fonctionne réellement — sans avoir à basculer vers Chrome. Avec en prime la sortie de preview de l'agent de modernisation C++ dans Visual Studio, ça a été une bonne semaine pour "laisser l'agent vérifier lui-même son travail".
