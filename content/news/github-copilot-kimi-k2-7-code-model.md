On July 1, GitHub made Kimi K2.7 Code generally available inside Copilot, alongside Claude, GPT, and Microsoft's own MAI-Code-1-Flash. That's four different labs' models sitting in one dropdown menu inside a single tool.

Whatever you think about the billing drama Copilot has had lately, this is a genuinely interesting bet: model choice as the actual product, rather than betting on one partner.
