El 1 de julio, GitHub puso a disposición general Kimi K2.7 Code dentro de Copilot, junto a Claude, GPT y el propio MAI-Code-1-Flash de Microsoft. Eso son cuatro modelos de laboratorios distintos en un solo menú desplegable de una sola herramienta.

Pienses lo que pienses del drama reciente con la facturación de Copilot, esta es una apuesta genuinamente interesante: la elección de modelo como el producto en sí, en lugar de apostar todo a un solo socio.
