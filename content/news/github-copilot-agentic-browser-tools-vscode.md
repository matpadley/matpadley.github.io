GitHub just shipped agentic browser tools to general availability in VS Code, letting Copilot navigate pages, inspect content, take screenshots, and actually validate that your web app works — without you tabbing over to Chrome. Paired with the C++ modernization agent graduating out of preview in Visual Studio, it's a solid week for "let the agent actually verify its own work."
