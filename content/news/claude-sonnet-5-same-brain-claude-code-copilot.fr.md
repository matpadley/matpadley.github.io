Claude Sonnet 5 est devenu disponible en version générale dans GitHub Copilot le 30 juin, juste après être devenu le modèle par défaut de Claude Code. Les premiers testeurs disent qu'il est nettement plus « agentique » que les anciennes versions de Sonnet, terminant des tâches que les modèles précédents auraient abandonnées, et vérifiant son propre travail sans qu'on le lui demande.

Donc si vous hésitiez entre Claude Code et Copilot en fonction de la qualité du modèle, ce choix vient de devenir beaucoup plus flou — vous risquez de faire tourner exactement le même modèle des deux côtés.
