Claude Sonnet 5 llegó a disponibilidad general en GitHub Copilot el 30 de junio, justo después de convertirse en el modelo predeterminado de Claude Code. Los primeros probadores dicen que es notablemente más "agéntico" que las versiones anteriores de Sonnet, terminando tareas que los modelos previos habrían abandonado, y revisando su propio trabajo sin que se lo pidan.

Así que si estabas eligiendo entre Claude Code y Copilot según la calidad del modelo, esa decisión se acaba de volver mucho más borrosa: podrías estar usando exactamente el mismo modelo en ambos casos.
