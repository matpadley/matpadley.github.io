La app de GitHub Copilot alcanzó disponibilidad general completa el 17 de junio, y es una verdadera aplicación de escritorio independiente, no solo una barra lateral de chat. Cada sesión levanta automáticamente su propio "worktree" de Git aislado, así que puedes correr varios agentes en paralelo sobre el mismo repo sin que se pisen entre sí, y una nueva función llamada "Agent Merge" puede llevar un pull request solo a través de CI, revisión y fusión si se lo permites.

Se siente como si GitHub estuviera admitiendo, calladamente, que la programación con agentes necesitaba su propia app, no solo otra extensión de IDE.
