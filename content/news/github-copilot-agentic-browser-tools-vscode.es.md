GitHub acaba de lanzar de forma estable las herramientas de navegación agéntica en VS Code, que permiten a Copilot recorrer páginas, inspeccionar contenido, tomar capturas de pantalla y comprobar que tu app web realmente funciona, sin tener que cambiar a Chrome. Sumado a que el agente de modernización de C++ salió de la fase de vista previa en Visual Studio, ha sido una buena semana para eso de "que el agente compruebe su propio trabajo".
