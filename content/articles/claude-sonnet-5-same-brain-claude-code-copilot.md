Claude Sonnet 5 went generally available inside GitHub Copilot on June 30, right on the heels of becoming the default model in Claude Code. Early testers say it's noticeably more agentic than past Sonnet versions, finishing tasks previous models would've given up on and double-checking its own work unprompted.

So if you've been picking Claude Code vs Copilot based on model quality, that decision just got a lot blurrier — you might be running the exact same model either way.
