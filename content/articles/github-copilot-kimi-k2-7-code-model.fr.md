Le 1er juillet, GitHub a rendu Kimi K2.7 Code disponible dans Copilot, aux côtés de Claude, GPT et le MAI-Code-1-Flash maison de Microsoft. Ça fait quatre modèles de labos différents dans un seul menu déroulant d'un seul outil.

Quoi qu'on pense du drame de la facturation récente de Copilot, c'est un pari vraiment intéressant : miser sur le choix du modèle comme véritable produit, plutôt que de tout miser sur un seul partenaire.
