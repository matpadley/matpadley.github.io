L'application GitHub Copilot est passée en disponibilité générale le 17 juin, et c'est une vraie application de bureau autonome, pas juste une barre latérale de chat. Chaque session lance automatiquement son propre « worktree » Git isolé, ce qui permet de faire tourner plusieurs agents en parallèle sur le même dépôt sans qu'ils se marchent dessus, et une nouvelle fonctionnalité « Agent Merge » peut faire avancer une pull request toute seule à travers la CI, la revue et la fusion si on la laisse faire.

On dirait que GitHub admet discrètement que le codage par agents avait besoin de sa propre appli, pas juste d'une extension d'IDE de plus.
