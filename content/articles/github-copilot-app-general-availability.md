The GitHub Copilot app went fully generally available on June 17, and it's a genuine standalone desktop app, not just a chat sidebar. Every session spins up its own isolated git worktree automatically, so you can run parallel agents on the same repo without them tripping over each other, and a new "Agent Merge" feature will drive a PR through CI, review, and merge on its own if you let it.

This feels like GitHub quietly admitting that "agent coding" needed its own app, not just another IDE extension.
