Claude Code cambió discretamente su modo de permisos predeterminado a "Manual" esta semana y desactivó la continuación automática en esos avisos `AskUserQuestion`. Traducción: las instalaciones nuevas, y muchas de las ya existentes, ahora se detendrán a preguntar antes de hacer cosas que antes hacían sin frenar.

Es claramente una jugada de seguridad ante todo un verano de titulares sobre agentes de código que se descontrolan a mitad de tarea. Si ya te habías acostumbrado a que Claude Code simplemente avanzara sin preguntar, prepárate para muchos más "¿puedo hacer esto?" a partir de ahora — igual puedes volver atrás con `--permission-mode` si preferías vivir peligrosamente.
