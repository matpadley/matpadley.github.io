We Buy Any Car required a system whereby dealers could view and bid, via online auctions, for vehicles from multiple suppliers (Manheim, BCA etc). Originally data was held in an Access database, which I migrated into SQL Server with an ASP.NET WCF layer using Dapper for the ORM. An MVC4 front end handled user authentication using ASP.NET membership providers.
