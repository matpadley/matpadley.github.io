We Buy Any Car avait besoin d'un système permettant aux concessionnaires de consulter et d'enchérir (via des ventes en ligne) sur des véhicules provenant de plusieurs fournisseurs (Manheim, BCA, etc.). Les données, initialement stockées dans une base Access, ont été migrées vers SQL Server avec une couche ASP.NET WCF utilisant Dapper comme ORM. Un front-end MVC4 gérait l'authentification des utilisateurs via les membership providers ASP.NET.
