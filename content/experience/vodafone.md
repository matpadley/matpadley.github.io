Initial role was to work as part of a small team to add new features to an SME app architected around BackboneJS, CQRS, Event Sourcing, RabbitMQ, RavenDB and SQL Server. This was re-worked into a WebAPI (including OWIN) back-end and Angular front-end. The domain layer was kept fairly simple but soon grew more complicated to work in.

The AngularJS portion of the project, which I introduced, was tested using Protractor tests (myself and another developer put this framework in place) and Jasmine within a TeamCity release pipeline.
